:lipstick: lo :lipstick:
===============

lo is a module of only was to simplify the console.log

## usage

```javascript
lo = require('lo')

lo.l('message')
```

### isConsoleLog param

true to output, false does not output

```javascript
lo = require('lo')

lo.isConsoleLog = false

lo.l('message')  // not output
```
